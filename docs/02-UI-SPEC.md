# 钓小龙虾小程序 UI 设计规范 v0.1

## 1. 设计目标

UI 需要表达：夏日、户外、轻松、亲子、年轻、有趣，同时保留地点决策产品的清晰效率。

关键词：

- 夏日感
- 小龙虾主题
- 亲子友好
- 地图探索
- 真实打卡
- 轻松可爱

## 2. 画布规格

微信小程序常用设计宽度：375px。

推荐 Figma Frame：

- iPhone 13 / 14：390 × 844
- 通用小程序稿：375 × 812

当前原型采用移动端 390px 宽视觉基准。

## 3. 色彩规范

### 主色

- 龙虾红：#F24E2E
- 夏日橙：#FF8A3D
- 草地绿：#35B46F

### 辅助色

- 水域蓝：#4AA8FF
- 阳光黄：#FFD166
- 米白背景：#FFF7EC
- 浅橙背景：#FFE8D8

### 中性色

- 标题黑：#1F2933
- 正文灰：#52606D
- 辅助灰：#9AA5B1
- 分割线：#E6E9ED
- 卡片白：#FFFFFF

### 语义色

- 成功：#35B46F
- 警示：#FFB020
- 错误：#E53935
- 信息：#4AA8FF

## 4. 字体规范

优先使用系统字体：

- iOS：PingFang SC
- Android：Noto Sans CJK / 系统默认

字号建议：

- 页面大标题：24px / 32px / SemiBold
- 模块标题：18px / 26px / SemiBold
- 卡片标题：16px / 24px / SemiBold
- 正文：14px / 22px / Regular
- 辅助信息：12px / 18px / Regular
- 标签文字：11px / 16px / Medium

## 5. 间距规范

- 页面左右边距：16px
- 卡片内边距：14px
- 模块间距：20px
- 卡片间距：12px
- 标签间距：6px
- 底部安全区：24px

## 6. 圆角规范

- 大卡片：20px
- 普通卡片：16px
- 按钮：14px
- 标签：999px
- 图片：14px
- 底部浮层：24px 顶部圆角

## 7. 阴影规范

卡片阴影：

- x: 0
- y: 8
- blur: 24
- color: rgba(242, 78, 46, 0.10)

地图浮层阴影：

- x: 0
- y: -8
- blur: 30
- color: rgba(31, 41, 51, 0.12)

## 8. 组件规范

### 8.1 顶部搜索栏

结构：
- 城市按钮
- 搜索输入框
- 定位图标

样式：
- 高度：44px
- 圆角：22px
- 背景：#FFFFFF

### 8.2 筛选胶囊

状态：
- 默认：白底灰字
- 选中：龙虾红底白字

字段：
- 附近
- 免费
- 亲子友好
- 出虾多
- 停车方便
- 可夜钓

### 8.3 钓点卡片

结构：
- 左侧图片
- 右侧名称
- 标签
- 距离
- 出虾指数
- 最近打卡

尺寸：
- 宽度：100%
- 图片：96 × 96
- 圆角：16px

### 8.4 虾况标签

等级：
- 爆护：红色
- 不错：橙色
- 一般：黄色
- 较少：灰色
- 未知：浅灰

### 8.5 今日虾况卡片

重点视觉模块，用浅橙渐变背景。

内容：
- 今日虾况等级
- 反馈人数
- 推荐时间
- 最近 7 天简化趋势

### 8.6 底部 Tab

Tab：
- 首页
- 地图
- 发布
- 我的

发布按钮可做突出样式，使用圆形加号。

## 9. 图标风格

建议使用线性圆角图标，局部填充主色。

核心图标：
- 小龙虾
- 地图定位
- 水波
- 收藏
- 分享
- 导航
- 拍照
- 儿童
- 停车
- 厕所
- 饵料

## 10. 页面视觉重点

### 首页
重点突出“今天去哪儿钓虾？”和附近推荐。

### 地图页
重点突出点位和底部浮层，不要让筛选信息遮挡地图。

### 详情页
重点突出“今日虾况”和“亲子安全”。

### 发布页
重点是降低表单压力，用选择题和卡片按钮代替长输入。
