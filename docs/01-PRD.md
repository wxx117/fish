# 钓小龙虾微信小程序 PRD v0.1

## 1. 产品概述

### 产品暂定名
虾友地图 / 去钓虾 / 小龙虾星球

### 一句话定位
帮助用户快速发现附近真实、好玩、安全、最近有人钓到小龙虾的地方。

### 核心价值
用户打开小程序后，应能快速回答：

1. 附近哪里能钓小龙虾？
2. 最近有没有人钓到？
3. 适不适合带小朋友？
4. 收不收费、环境怎么样？
5. 怎么导航过去？

## 2. 目标用户

### P0：亲子家庭
- 诉求：周末带娃户外玩，安全、干净、方便停车。
- 决策因素：安全、厕所、停车、收费、最近虾况。

### P0：年轻人 / 情侣 / 学生
- 诉求：暑期找有趣活动，可拍照、可分享、可约朋友。
- 决策因素：好玩、出片、距离近、最近有人去。

### P1：钓虾达人
- 诉求：分享钓点、交流饵料、记录战绩。
- 决策因素：出虾率、时间段、水域情况。

### P2：商家
- 诉求：获客、展示环境、发布活动。
- 决策因素：曝光、评价、转化。

## 5. 核心用户流程

### 找钓点流程
打开小程序 → 定位城市 → 查看附近推荐 → 筛选标签 → 进入详情 → 查看今日虾况 → 导航前往。

### 打卡流程
进入钓点详情 → 点击“我来打卡” → 填写钓获数量、虾况、饵料、时间段 → 上传照片 → 发布。

### 发布钓点流程
点击发布 → 选择“发布钓点” → 地图定位 → 填写基础信息 → 上传照片 → 提交审核。

## 6. 页面需求

### 6.1 首页

#### 页面目标
让用户快速发现附近值得去的钓虾地点。

#### 主要模块
1. 城市定位与搜索
2. 快捷筛选：附近、免费、亲子、出虾多、停车方便
3. 今日虾况榜
4. 亲子友好推荐
5. 最新打卡
6. 钓点卡片列表

#### 钓点卡片字段
- 图片
- 名称
- 距离
- 出虾指数
- 亲子标签
- 收费信息
- 最近打卡时间

### 6.2 地图页

#### 页面目标
让用户基于位置快速发现钓点。

#### 主要模块
1. 地图
2. 顶部搜索
3. 筛选胶囊
4. 点位标记
5. 底部钓点浮层卡片

#### 点位类型
- 免费野钓点
- 收费钓场
- 农家乐
- 待确认钓点

### 6.3 钓点详情页

#### 页面目标
帮助用户判断这个钓点是否值得去。

#### 主要模块
1. 顶部图片
2. 名称、评分、距离
3. 导航、收藏、分享
4. 核心标签
5. 今日虾况
6. 基础信息
7. 设施信息
8. 用户打卡
9. 避坑提醒
10. 附近推荐

### 6.4 发布打卡页

#### 页面目标
让用户在 1 分钟内完成真实反馈。

#### 字段
- 钓到数量
- 出虾情况
- 推荐时间段
- 使用饵料
- 是否适合小朋友
- 环境评分
- 安全评分
- 上传照片
- 避坑提醒

### 6.5 发布钓点页

#### 页面目标
鼓励用户贡献新地点，同时确保数据质量。

#### 字段
- 钓点名称
- 地图定位
- 钓点类型
- 是否收费
- 收费说明
- 是否适合亲子
- 是否停车方便
- 是否有厕所
- 上传照片
- 补充说明

### 6.6 我的页面

#### 页面目标
沉淀用户行为和贡献。

#### 模块
- 用户资料
- 我的收藏
- 我的打卡
- 我的发布
- 我的积分
- 反馈建议

## 7. 核心指标

### 北极星指标
每周有效钓点访问决策次数。

### 关键指标
1. 首页到详情点击率
2. 详情到导航点击率
3. 收藏率
4. 分享率
5. 打卡发布率
6. 新钓点提交数
7. 审核通过钓点数
8. 周末活跃用户数

## 8. 风险与审核

### 风险
1. 用户误入私人鱼塘
2. 野外水域安全风险
3. 儿童靠近水域风险
4. 虚假钓点和刷评价
5. 小龙虾食用安全问题

### 处理策略
1. 新钓点审核后展示
2. 野钓点强化安全提示
3. 支持举报无效地点
4. 私人区域不展示精确入口
5. 显示“用户分享，出行前请自行确认”提示
