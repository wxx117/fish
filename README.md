# 钓小龙虾微信小程序产品与 UI 原型

这是一个用于“钓小龙虾地点分享社区”的微信小程序 MVP 项目，已经按现有 UI 原型风格落地为原生微信小程序结构。

## 当前产出

- 产品 PRD：[docs/01-PRD.md](docs/01-PRD.md)
- UI 设计规范：[docs/02-UI-SPEC.md](docs/02-UI-SPEC.md)
- 静态高保真方向原型：[prototype/index.html](prototype/index.html)
- 微信小程序入口：app.json、app.js、app.wxss
- 微信小程序页面：pages/

## 产品定位

帮助用户快速发现附近真实、好玩、安全、最近有人钓到小龙虾的地方。

## MVP 核心页面

1. 首页
2. 地图页
3. 钓点详情页
4. 发布打卡页
5. 发布钓点页
6. 我的页面

## 微信小程序开发

使用微信开发者工具导入当前目录即可：

1. 打开微信开发者工具
2. 选择“导入项目”
3. 项目目录选择当前文件夹
4. AppID 可先选择测试号或使用 touristappid
5. 编译运行

## 当前小程序能力

- 首页推荐钓点
- 地图风格钓点探索页
- 钓点详情页
- 导航、收藏、分享入口
- 发布打卡表单
- 发布新钓点表单
- 我的页面
- 本地模拟数据
